# The 52-Week Culinary Project

A year-long plan to turn daily cooking into a self-guided culinary school. This repo provides the structure, templates, and a 52-week progression so you can plan dinners that align with each weekly topic.

## How to use

1. Start with the curriculum map in [docs/02-curriculum-map.md](docs/02-curriculum-map.md).
2. For each week, copy the template in [docs/03-week-template.md](docs/03-week-template.md) into [weeks/](weeks/) and fill it in.
3. Use the day template in [docs/04-day-template.md](docs/04-day-template.md) to plan weeknight dinners (30-45 minutes) and a major Saturday meal.
4. Track notes and reflections in [logs/README.md](logs/README.md).

## Weekly rhythm (default)

- Mon-Fri: 30-45 minute dinners aligned to the weekly topic.
- Sat: 1 major meal that integrates the week's core techniques.
- Sun: optional prep, leftovers, or a light skill drill.
