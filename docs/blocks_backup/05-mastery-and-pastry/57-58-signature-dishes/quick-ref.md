## Wok Hei: What It Is and How to Get It

| Factor | Requirement | Common Failure |
|---|---|---|
| Heat | Maximum burner output — gas preferred | Electric/induction: insufficient BTU |
| Pan | Seasoned wok; screaming hot before oil | Cold wok = steaming, not searing |
| Ingredients | Dry surface; small batches | Wet veg/protein releases steam |
| Motion | Constant toss; 30–60 seconds max | Over-stirring = temperature drop |
| Oil | High smoke-point (peanut, grapeseed) | Olive oil → burns bitter |
| Sequence | Aromatics → protein → veg → sauce | Wrong order = mushy or raw |

---

## Wok Seasoning Method

1. Heat wok over high flame until water droplets vanish instantly
2. Add 1 tbsp high-smoke-point oil; swirl to coat
3. Rub with folded paper towel (tongs — do NOT burn yourself)
4. Repeat 3× until surface is uniformly darkened
5. Store dry; never soap-scrub; re-season if rust appears

## Mise en Place Workflow

**Before you cook:**
1. Read the full recipe twice
2. Write a mise en place list (every ingredient prepped to what state)
3. Group by timing: what cooks longest goes first
4. Set up stations: cutting board, stove, plating area

**While cooking:**
- Keep mise en place bowls/plates in use order left to right
- Clean as you go — wipe board between tasks
- Taste as you build — every layer, not just the end

**For a 3-course dinner (timing template):**
| Stage | Window | Tasks |
|---|---|---|
| T-2 hours | Active prep | Stock, braises, doughs, marinades, dessert |
| T-1 hour | Component prep | All veg cut, proteins seasoned, sauces made |
| T-30 min | Final staging | Plates warmed, sauces reheated, garnish ready |
| T-0 | Service | Cook-to-order only; assemble components |

## Chocolate Chip Cookie Variables (Decision Chart)

| Preference | Adjustment |
|---|---|
| Crispier edge | More white sugar; higher temp (375°F); thinner dough ball |
| Chewier center | More brown sugar; bread flour; underbake slightly |
| More complex flavor | Brown butter; overnight rest (minimum); flaky salt on top |
| Crunchy throughout | Lower temp (325°F); longer bake; fully cool before eating |
| [Deeper chocolate](https://www.seriouseats.com/the-food-lab-best-chocolate-chip-cookie-recipe) | [Chopped dark chocolate instead of chips; chips melt different](https://www.seriouseats.com/the-food-lab-best-chocolate-chip-cookie-recipe) |
| [Bigger pools of chocolate](https://www.seriouseats.com/the-food-lab-best-chocolate-chip-cookie-recipe) | Chopped wafers; reserve chunks for pressing on top before baking |

## Caramel Trouble Guide

| Problem | Cause | Fix |
|---|---|---|
| Crystallized / grainy | Sugar crystals on pan sides got in | Wet brush pan sides; start over |
| Didn't get dark enough | Under-cooked; light amber isn't enough | Go to deep amber (350°F+) |
| Bitter / burnt | Over-cooked past 370°F | Start over |
| Seized when cream added | Cream too cold; rapid temp drop | Warm cream slightly first |
| Too thick when cool | Too little cream | Add more warm cream and stir |

## Signature Dish Evaluation Criteria

When deciding if a dish is "yours," ask:

| Question | Yes / No |
|---|---|
| Can I make this without a recipe? | |
| Have I made this 3+ times? | |
| Do I know what can go wrong and how to fix it? | |
| Have I adjusted it to my own taste? | |
| Would I serve this at a dinner party without anxiety? | |

**3+ Yes answers:** This is likely a signature dish for you.
