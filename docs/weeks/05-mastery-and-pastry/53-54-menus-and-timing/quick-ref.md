## Dinner Party Menu Composition Rules

| Principle | Application |
|---|---|
| Weight before pleasure | Light → medium → rich → light (dessert) |
| Acid punctuation | One acid-forward dish after each rich one |
| Color contrast | A plate of beige food needs green; a plate of all green needs contrast |
| Temperature variety | Not all hot; not all cold; one warm, one room temp |
| Avoid repetition | No protein appears twice; no sauce repeats |
| Local and seasonal | The best menus are limited by season, not ambition |

## Make-Ahead Lead Times

| Component | How Far Ahead | Storage |
|---|---|---|
| Braised meats | 1–2 days | Refrigerate in braising liquid; reheat gently |
| Stocks / sauces | 3–4 days | Refrigerate; reduce and finish day-of |
| Pasta dough (uncut) | 1 day | Wrapped airtight; refrigerate |
| Pastry dough | 2–3 days | Refrigerate; blind-bake on party day |
| Desserts (custard-based) | 1 day | Refrigerate; brûlée / garnish day of |
| Desserts (gelatin) | 1–2 days | Refrigerate unmolded; garnish last minute |
| Tarts and pastry | 1 day (baked) | Room temp; fill day of |
| Charcuterie / cured meats | 1 week | Refrigerate; slice day of |
| Pickled vegetables | 2 days+ | Refrigerate; use as needed |
| Bread (baked) | Day before | Room temp wrapped; reheat 5 min at 350°F |

## Recipe Scaling Guide

| Element | Scales Linearly | Scales Non-Linearly | Notes |
|---|---|---|---|
| Dry ingredients | ✓ | [—](https://www.seriouseats.com/how-to-set-up-a-station-like-a-pro) | Simple multiplication |
| Wet ingredients (sauces) | ✓ | [—](https://www.seriouseats.com/how-to-set-up-a-station-like-a-pro) | Double sauce = double quantity |
| Salt and spices | [—](https://www.seriouseats.com/how-to-set-up-a-station-like-a-pro) | ✓ | Start at 75%; taste and adjust |
| Acid (lemon, vinegar) | [—](https://www.seriouseats.com/how-to-set-up-a-station-like-a-pro) | ✓ | Less than linear; easy to over-acid |
| Aromatics (garlic) | [—](https://www.seriouseats.com/how-to-set-up-a-station-like-a-pro) | ✓ | [1.5× for 2× recipe](https://www.tastingtable.com/1047393/tips-for-scaling-recipes-up-or-down/) |
| Baking time | [—](https://www.seriouseats.com/how-to-set-up-a-station-like-a-pro) | ✓ | Same; watch internal temp not clock |
| Braising / roasting time | [—](https://www.seriouseats.com/how-to-set-up-a-station-like-a-pro) | ✓ | Size of piece, not number of pieces |
| Pan size | [—](https://www.seriouseats.com/how-to-set-up-a-station-like-a-pro) | ✓ | Bigger pan → less crowding → better browning |

## Wine and Beverage Pairing Basics

| Dish Character | Wine Style | Why |
|---|---|---|
| Oysters / raw shellfish | Crisp Muscadet or Chablis | High acid mirrors brine |
| Rich cream sauces | White Burgundy or Viognier | Weight matches weight |
| Grilled white fish | Sauvignon Blanc, Vermentino | Acid cuts richness, herbal notes |
| Braise / short rib | Cabernet Sauvignon, Syrah | Tannin + protein = balance |
| Duck / duck confit | Pinot Noir; light Grenache | Less tannin for fat richness |
| Lamb | Rhône red; Ribera del Duero | Iron in lamb → earthy reds |
| Spicy food | Off-dry Riesling; Gewürztraminer | Residual sugar cools heat |
| Charcuterie / cured | Chenin Blanc; light Beaujolais | Acid cuts fat |
| Rich dessert | Sauternes; Banyuls; Tawny Port | Wine sweeter than dish |
| Chocolate | Banyuls; Amontillado Sherry | Bitter matches bitter |

**Non-alcoholic alternatives:**
- Acid-forward dishes → shrub (vinegar-based) with sparkling water
- Rich main courses → strong sparkling water with lemon
- Dessert → good espresso or brewed cold brew

## Service Flow Checklist

**Before guests arrive:**
- [ ] Table set (plates, silverware, glasses, napkins, candles)
- [ ] Canapés plated and covered
- [ ] Dessert in refrigerator, plated or ready to plate
- [ ] All mise en place verified; nothing left unprepped
- [ ] Oven preheated
- [ ] Sauces in bain-marie or holding position

**During service:**
- [ ] Clear each course before next appears
- [ ] Warm plates before serving hot courses
- [ ] Pour before clearing (don't rush)
- [ ] Final sauce/garnish added at last moment

## Kitchen Triage: When Things Go Wrong

| Problem | Immediate Fix |
|---|---|
| Sauce broken (butter) | Whisk ice cube in; or start from emulsified base |
| Overcooked protein | Thin-slice and serve with assertive sauce; never apologize at the table |
| Underseasoned dish | Salt at the table (offer Maldon); squeeze lemon on anything savory |
| Sticking / burning stuck | [Deglaze with wine or stock; incorporate the fond into sauce](https://www.seriouseats.com/guide-to-cooking-with-wine-11921603) |
| Dessert didn't set | Serve in glasses as "deconstructed"; add texture on top |
| Completely failed component | Fill with charcuterie, cheese, bread, olives — always keep these in reserve |
