# Serious Eats: The 52-Week "Food Lab" Curriculum

This curriculum is designed to transform a novice into a technician using the scientific principles of J. Kenji López-Alt, Stella Parks, and the Serious Eats team.

**Schedule:**
- **Mon-Wed, Fri:** 15m Learning / 45m Cooking (Full Meals)
- **Thursday:** Light Technique / Light Meal
- **Saturday:** Benchmark Meal (3-4 Hours)
- **Sunday:** Recovery / Light Meal
